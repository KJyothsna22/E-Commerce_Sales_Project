# E-Commerce_Sales_Project

## Overview
This project presents an interactive dashboard built using Excel to analyze e-commerce sales data. It helps in understanding key business metrics, trends, and performance insights.

## Objective
The goal of this project is to:

1. Analyze sales performance
2. Identify top products and categories
3. Track revenue trends over time
4. Support data-driven decision making

## Tools Used
1. Microsoft Excel
2. Pivot Tables
3. Charts & Graphs
4. Slicers & Filters

## Key Insights
1. Monthly and yearly sales trends
2. Top-performing products and categories
3. Customer purchase behavior
4. Region-wise sales distribution
